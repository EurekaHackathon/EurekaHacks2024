Eureka Hacks 2024 Website built with Next.js

## Running locally

1. Clone the repo

```bash
$ git clone https://github.com/EurekaHackathon/EurekaHacks2024.git
```

2. Install dependencies and run

```bash
$ npm i
$ npm run dev
```
