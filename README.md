# EurekaHacks2024
Eureka Hacks 2024 Website

## Running locally

1. Clone the repo

```bash
$ git clone https://github.com/Eureka.git
```

2. Install dependencies and run

```bash
$ npm install
$ npm start
```

To build for production and test response times you can use the following

```bash
$ npm run build
$ serve -s build
```
